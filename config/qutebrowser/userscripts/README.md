# Qutebrowser user scripts

Note: Copy these to `~/.local/share/qutebrowser/userscripts/`

## Readability

```sh
pip install -r requirements.txt
chmod +x readability
```

Command:

```
:spawn --userscript readability
```

## Qutepocket

```sh
pip3 install pocket-api --user
```

Command:

```
pip3 install pocket-api --user
```
